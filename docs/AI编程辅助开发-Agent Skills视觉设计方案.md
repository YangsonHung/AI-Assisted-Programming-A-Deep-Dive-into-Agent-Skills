# AI编程辅助开发：Agent Skills 视觉设计方案

## 1. 设计理念

### 整体视觉方向

- 整体视觉呈现一种精准、思辨、前沿的氛围。设计灵感源自《MIT Technology Review》的严谨版式与电影《降临》中人类与外星智慧沟通时的界面美学，强调信息的结构化与逻辑性。核心理念是通过冷静、深邃的视觉语言，营造一个专注、理性的思考空间，引导观众深入理解AI辅助编程的本质，从执行者到指挥者的角色进化。

### 内容密度

- 内容密度等级为中高密度。专为技术分享和知识传递设计，每页承载3-5个核心要点，在保证信息清晰传达的同时，通过约15%-20%的留白，营造出专业、严谨的视觉节奏，避免信息过载。

## 2. 色彩系统

**主色调**

- 背景色：#1A1D21（深邃墨蓝，营造专注氛围）
- 主色：#E8E8E8（浅灰白，主要信息承载）
- 次要色：#8A8F98（中性灰，辅助信息区分）
- 强调色：#34D399（青绿色，关键要点突出）

**文字色彩**

- 深色文字（用于浅色背景）：#1A1D21
- 浅色文字（用于深色背景）：#E8E8E8

## 3. 字体设计

**标题字体**

- 中文字体
  - Chiron Hei HK（现代几何黑体，线条锐利，科技感突出，契合AI主题）
- 英文字体
  - Hedvig Letters Sans（个性鲜明，带有人工设计感，呼应AI的创造性）

**正文字体**

- 中文字体
  - Noto Sans SC（结构规范，风格中性，确保长文案的最高可读性）
- 英文字体
  - Quattrocento Sans（经典优雅，温和易读，适合大段英文内容的呈现）

## 4. 图像风格

- 图像风格以抽象、概念化为主。优先使用高质量的3D渲染图形，表现数据流、神经网络节点或抽象代码结构，材质可参考磨砂玻璃或全息投影效果。摄影图片则选择具有建筑感、光影对比强烈的黑白或低饱和度图像，聚焦于结构、线条与光影，避免使用具象的人物或场景照片，以保持整体的科技感与思辨氛围。
